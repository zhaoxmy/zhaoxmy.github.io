A - Not Too Hard

原题链接:A-Not Too Hard

难度：入门

算法：无

题解：

要求解n个数里面所有小于等于x的数的和,逐个求解即可，不加赘述

AC code

#include <iostream>
using namespace std;
int main(){
    int n,x;
    cin>>n>>x;
    long long ans=0;
    int a=0;
    for(int i=0;i<n;i++){
        cin>>a;
        if(a<=x)
            ans+=a;
    }
    cout<<ans;
}

B - 11/11

原题链接:B - 11/11

难度：入门

算法：无

题解：

数据范围最多有100个月，每个月最多100天，暴力枚举每一天即可。
比较麻烦的是如何判断一个日期中每一个整数都相同。我的方案是利用tostring函数转换为字符串，然后判断。

AC code

#include <iostream>
#include <vector>
using namespace std;
bool ck(string b){
    char te=b[0];
    for(int i=1;i<b.length();i++)
        if(b[i]!=te)
            return 0;
    return 1;
}
int main(){
    int n;
    cin>>n;
    vector<int> days(n);
    for(int i=0;i<n;i++)
        cin>>days[i];
    int ans=0;
    for(int i=1;i<=n;i++){
        for(int j=1;j<=days[i-1];j++)
            if(ck(to_string(i)+ to_string(j)))
                ans++;
    }
    cout<<ans;
}

C - Consecutive

原题链接:C - Consecutive

难度：初级

算法：前缀和

题解：

前缀和的模板问题，考虑声明一个数组v，表示字符串s相应位置的贡献。
具体的说，若sp==sp+1,
则vp=1，
否则vp0。
问题转换为求解数组v在区间[l,r)上的和，典型前缀和问题

AC code

#include <iostream>
#include <vector>
using namespace std;
template<typename T>
void reverseDifference(vector<T> &a){
    for(int i=1;i<a.size();i++)
        a[i]+=a[i-1];
}
int main(){
    int n,q;
    cin>>n>>q;
    string a;
    cin>>a;
    vector<int> v(n+1);
    for(int i=0;i<n-1;i++)
        if(a[i]==a[i+1])
            v[i+1]++;
    reverseDifference(v);
    int l,r;
    for(int i=0;i<q;i++){
        cin>>l>>r;
        cout<<v[r-1]-v[l-1]<<endl;
    }
}

D - Take ABC

原题链接:D - Take ABC

难度：初级

算法：顺序表

题解一

题目要求删除所有连续的ABC，一个可行的思路是：每遇到一次c，就检查他的前两位是不是a,b，如果是，则将其删除，复杂度瓶颈在于数组删除所需o(n)时间，n次就需要n2。因为复杂度瓶颈在于删除，所以很显然可以考虑链表。
在实现上，由于直接链表中删除节点会导致对应的迭代器失效。所以实现为对于每一个节点，检查他的前三个是不是c,b,a,若是，则删除前三个。
由于list不支持访问前三个，只支持访问前一个，所以重载了list迭代器的加法和减法运算符，以便代码实现。

AC code

#include <list>
#include <iostream>
using namespace std;
template<typename T>
_List_iterator<T> operator+(_List_iterator<T> a, int x){
    for(int i=0;i<x;i++)
        a++;
    return a;
}
template<typename T>
_List_iterator<T> operator-(_List_iterator<T> a, int x){
    for(int i=0;i<x;i++)
        a--;
    return a;
}
int main(){
    string a;
    cin>>a;
    list<char> li;
    for(char i : a)
        li.push_back(i);
    list<char>::iterator cd;
    cd=li.begin();
    cd=cd+3;
    for(;cd!=li.end();cd++){
        if(*(cd-1)=='C' && *(cd-2)=='B' && *(cd-3)=='A')
            li.erase(cd-3,cd);
    }
    if(*(cd-1)=='C' && *(cd-2)=='B' && *(cd-3)=='A')
        li.erase(cd-3,cd);
    for(auto i:li)
        cout<<i;
}

题解二

赛后拜读大佬代码，得到了一种新的解题思路。
首先我们知道，每次删除末尾的C字母下标是单调递增的，那我们完全可以维护一个vector，如果队列尾部三个分别是’A’’B’’C’则弹出队列末尾三个，然后不断将s加入vector的末位。最终得到的vector即为答案。

AC code

#include <iostream>
#include <vector>

using namespace std;
int main(){
    string a;
    cin>>a;
    vector<char> v;
    for(auto i:a){
        v.push_back(i);
        if(v.size()>=3 && *(v.end()-1)=='C' && *(v.end()-2)=='B' && *(v.end()-3)=='A')
            for(int w=0;w<3;w++)
                v.pop_back();
    }
    for(auto i:v)
        cout<<i;
}

D-总结

虽然D只是一道初级的顺序表问题，但在却能在一定程度上考验选手对不同顺序表的理解能力。题解二的写法仅仅只有16行，而且仅需要题解一三分之一的运行时间
作为最广为人知的stl容器，vector对于任何基于线性表的算法都有着优秀的适配能力，如果仅仅把vector视作动态数组，你将会失去c++的一大宝藏
